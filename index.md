# My Learning Journey Across Javascript, GitHub, and Claude Code

## Javascript Fundamentals via Zenva

JavaScript forms the foundation of modern web development. Through Zenva's JavaScript 101 course, I am learning the core language features that power interactive web applications. This structured approach builds understanding from basic syntax through to practical programming concepts.

### Core Language Concepts

The curriculum covers essential JavaScript mechanics that every developer needs to master. Variables, data types, and operators form the bedrock of the language. Control flow structures like conditionals and loops enable dynamic program behavior. Functions serve as the building blocks for organized, reusable code.

- [ ] Master conditional statements (if, else, switch)
- [ ] Practice loop structures (for, while, do-while)

## GitHub Skills Development

GitHub Skills provides hands-on tutorials that teach version control and collaborative development practices. Learning GitHub directly on the platform creates immediate connection between concepts and practical workflow. This approach grounds abstract version control principles in real repository management.

### Collaboration and Version Control

Version control systems enable teams to work together on codebases without losing work or creating conflicting changes. GitHub serves as both the platform and the curriculum for understanding how to manage code history, create branches, and contribute changes through pull requests.

#### Progress Checklist

- [ ] Complete GitHub introduction tutorials
- [ ] Practice creating repositories and commits
- [ ] Learn branching strategies and workflows
- [ ] Understand pull request review processes
- [ ] Practice collaborative contribution patterns
- [ ] Explore GitHub issues and project management
- [ ] Master merge conflict resolution

## Claude Code Development via Traversaal

The Claude Code in Practice course at Traversaal teaches AI-assisted development using Claude as a collaborative coding partner. This modern approach shifts how developers think about problem-solving and implementation. Claude Code tools enable faster iteration and help validate architectural decisions before committing to complex implementations.

### Building with AI-Assisted Development

Working with Claude Code changes the development experience by allowing natural language problem description to generate and refine code. The first module covers skill creation and prompt design, teaching how to express requirements clearly enough for an AI to understand implementation intent.
