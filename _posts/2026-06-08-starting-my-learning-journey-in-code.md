# Beginning Three Parallel Learning Paths

Today marks the start of my journey into programming and development. I am taking on three interconnected courses that build different but complementary skills. This blog will document my progress, insights, and breakthroughs as I learn.

## Why These Three Paths?

Learning programming requires more than just syntax. I need to understand how to write code, how to collaborate with others on code, and how to leverage modern tools like AI to work more effectively. These three courses address each of those needs.

JavaScript through Zenva gives me the foundational language skills I need to build interactive applications. GitHub Skills teaches me version control and the collaborative workflows that real development teams use every day. Claude Code through Traversaal introduces me to AI-assisted development, showing me how to work with AI as a coding partner rather than writing everything from scratch.

## What I'm Learning This Week

My focus right now is on building a solid foundation in each area without trying to rush ahead. In JavaScript, I am working through core language concepts like variables, data types, and control flow. On GitHub, I am learning how repositories work and how to make my first commits. With Claude Code, I am studying the architecture of AI-assisted development and understanding how to express requirements clearly.

## The Connection Between These Skills

What excites me most is how these three areas reinforce each other. As I learn JavaScript fundamentals, I will practice version control on GitHub to manage my code. As I build projects, I will use Claude Code to accelerate my development and focus on understanding concepts rather than getting stuck on syntax. Each tool and skill makes the others more effective.

## Next Steps

Over the coming weeks, I plan to build small projects that integrate all three skills. I will write JavaScript code, commit it to GitHub repositories, and use Claude Code to help me solve problems and iterate quickly. This blog will be my space to reflect on what I am learning and share insights with others on a similar journey.

Stay tuned for updates on my progress!
